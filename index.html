Læringsportal oversikt

HTML

HTML er grunnspråket for å lage nettsider. Det står for HyperText Markup Language, og brukes til å strukturere innholdet på en webside slik at nettleseren vet hva som er overskrifter, avsnitt, bilder, lenker osv.

Les mer

CSS

CSS (Cascading Style Sheets) er språket som brukes for å bestemme utseendet på en nettside. Det styrer farger, størrelser, avstander, layout og mye mer.

Les mer

JavaScript

JavaScript (JS) er språket som gjør nettsider interaktive.
