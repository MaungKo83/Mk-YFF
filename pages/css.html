CSS‑koden

📘Hva er CSS‑koden?

Denne CSS‑filen styrer utseende, layout og design for hele nettsiden.

... Den er delt inn i logiske seksjoner for variabler, reset, layout, header, hero‑seksjon, kort (cards), skjema, footer og responsiv design.



🎨 1.CSS‑variabler (:root)

Her defineres globale fargevariabler.
... Fordeler:
enklere vedlikehold (endrer du én verdi → oppdateres alle steder)
konsistent design
enklere å lese koden
Eksempel:
var(--brand) brukes som hovedfarge for knapper og lenker.



🔧 2.CSS Reset

Denne delen nullstiller nettleserens standardstiler. ... Formål: sikre lik oppførsel i alle nettlesere
hindre uønsket margin/padding
gjøre layout mer forutsigbart



🖥️ 3.Grunnstil for body

Dette setter: ...

moderne systemfont for bedre lesbarhet
hvit tekst mot mørk bakgrunn
gradient bakgrunn som gir et profesjonelt preg
linjeavstand for bedre lesing



📏4. Container – breddebegrensning

Denne sørger for at innholdet: ...

aldri blir bredere enn 1100px
har marg på begge sider
er sentrert



♿ 5. Skip‑link (tilgjengelighet)

Dette gir bedre tilgjengelighet: ...

skjermlesere og tastaturnavigasjon kan hoppe rett til innholdet
vises kun når brukeren trykker Tab



🧭 6. Header og navigasjon/h3>

Effekt: ...

headeren følger med når man scroller
semi‑transparent med «glass‑effekt»

Navigasjon

Navigasjon: ...

menyer ligger horisontalt
hover‑effekt
aktuell side markeres med aria-current="page"



Mobilmeny

Gir: ...

hamburger‑meny på mobil
meny åpnes/lukkes med JavaScript



7. Hero‑seksjonen (forsidebilde + tekst)

Hero‑seksjonen:Gir god luft rundt hero‑seksjonen. ...

Grid-layout: venstre: tekst
høyre: bilde
høyre kolonne har fleksibel bredde (fra 280px til 520px)



Responsive Hero

Når skjermen er liten: ...

bildedelen havner under teksten
alt blir midtstilte kolonner



Hero-bilde

Dette gir: ...

myke avrundede hjørner
moderne skygge
bilde som holder samme proporsjoner på alle sider
ingen forvrenging



📚 8. Cards‑seksjon

Gir tre responsive kolonner. ...

moderne kortdesign
skygge for dybde
mørkt panel med kontrast mot tekst



📬 9. Kontaktform

Kontaktform: ...

bruker grid‑layout
mørk bakgrunn + lys tekst
blå outline for tilgjengelig fokus



🦶 10. Footer/h3>

Footer skiller seg tydelig fra hovedinnholdet med: ...

topp-border
grå tekst
godt med luft

Dette gjør at layouten fungerer på: ...

PC
nettbrett
mobil
